What this demo shows

  • Natural-language robot control
    Send robot commands through a clean web interface.
  • Semantic place navigation
    Navigate to grounded room-level targets in the small-house environment.
  • ROS 2 and Nav2 execution
    Bridge web requests to live robot actions, localization, and control.
  • Vision-language perception
    Use SAM + CLIP-style perception for scene summary and object awareness.
  • Full-stack architecture
    FastAPI backend, Next.js frontend, ROS 2 integration, and YAML grounding.
